给出集合 [1,2,3,…,n]，其所有元素共有 n! 种排列。

按大小顺序列出所有排列情况，并一一标记，当 n = 3 时, 所有排列如下：

```
"123"
"132"
"213"
"231"
"312"
"321"
```

给定 n 和 k，返回第 k 个排列。

说明：

给定 n 的范围是 [1, 9]。

给定 k 的范围是[1,  n!]。

示例 1:

```
输入: n = 3, k = 3
输出: "213"
```

示例 2:

```
输入: n = 4, k = 9
输出: "2314"
```

## DFS

O(K * O(N)), 其中K = N!

依次查找，直到找到第K个组合，然后返回

```python
class Solution:
    def getPermutation(self, n: int, k: int) -> str:
        self.res = ''
        self.count = 0
        self.helper(n, k, '', set())
        return self.res
        
    def helper(self, n: int, k: int, ans: str, visited: set) -> None:

        if self.count >= k:
            return
        
        if len(ans) == n:
            self.count += 1
            if self.count == k:
                self.res = ans
            return
        
        for i in range(1, n + 1):
            if i in visited:
                continue

            visited.add(i)
            self.helper(n, k, ans + str(i), visited)
            visited.remove(i)
```
