## 67. 二进制求和 

给你两个二进制字符串，返回它们的和（用二进制表示）。

输入为 非空 字符串且只包含数字 1 和 0。

示例 1:

```
输入: a = "11", b = "1"
输出: "100"
```

示例 2:

```
输入: a = "1010", b = "1011"
输出: "10101"
```
 
提示：

```
每个字符串仅由字符 '0' 或 '1' 组成。
1 <= a.length, b.length <= 10^4
字符串如果不是 "0" ，就都不含前导零。
```

## 模拟加

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        
        carry = 0
        i = len(a) - 1
        j = len(b) - 1
        res = ''
        
        while i >= 0 or j >=0 or carry > 0:
            
            left = int(a[i]) if i >=0 else 0
            right = int(b[j]) if j >=0 else 0        
            val = left + right + carry
    
            res = str(val % 2) + res
            carry = val // 2
            
            i -= 1
            j -= 1
            
        return res
```

二进制重写

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        
        carry = 0
        i = len(a) - 1
        j = len(b) - 1
        res = ''
        
        while i >= 0 or j >=0 or carry > 0:
            
            left = int(a[i]) if i >=0 else 0
            right = int(b[j]) if j >=0 else 0        
            val = left + right + carry
    
            res = str(val & 1) + res # val % 2 + res
            carry = val >> 1 # val // 2
            
            i -= 1
            j -= 1
            
        return res
```
